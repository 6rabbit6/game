# 赛事系统

这是一个纯前端的赛事展示与录入系统，直接打开 `index.html` 即可使用。

## 功能

- 首页展示赛事总数、进行中状态和赛事列表
- 赛事详情页展示按比赛日拆分的赛程表
- 分组详情页展示某个项目的分组和运动员名单
- 后台管理支持修改站点文案、赛事信息、赛程、分组和运动员
- 数据自动保存到浏览器 `localStorage`
- 支持导入、导出 JSON 数据

## 使用方式

1. 直接双击 `index.html`，或在当前目录运行本地静态服务。
2. 进入“后台管理”录入或修改赛事内容。
3. 修改后会自动保存，刷新页面不会丢失。
4. 如需迁移数据，可在后台导出 JSON，再在其他设备导入。
